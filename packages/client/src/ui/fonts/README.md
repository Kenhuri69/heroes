# Fontes locales

- `cinzel-latin.woff2` — **Cinzel** (fonte variable, poids 400–900), sous-ensemble
  latin, 25,9 Ko. Voix « display » du design system (UXD-1) : titres, gros
  boutons, chiffres de combat — capitales trajanes, la voix des titres HoMM.
- Licence : **SIL Open Font License 1.1** — © The Cinzel Project Authors
  (https://github.com/NDISCOVER/Cinzel). Téléchargée depuis Google Fonts
  (fonts.gstatic.com, Cinzel v26) et **servie localement** (aucune requête
  CDN à l'exécution — CSP/immersion offline).
- Le corps de texte reste `system-ui` (lisibilité) : la display ne porte que
  les titres/actions courtes (Cinzel n'a pas de vraies minuscules — petites
  capitales).
